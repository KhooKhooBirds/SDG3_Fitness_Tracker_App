# User Prototype Feedback Analysis
 
## Overview / Analysis Summary
 
The survey for the first fitness app prototype, nicknamed **TRACKDEEZSTATS**, was conducted from **March 9–10**. The prototype was a result of the first sprint cycle, informed by the Requirement Analysis conducted earlier.
 
- **Total responses:** 14 (all from initial Requirement Gathering participants)
- **Overall sentiment:** Positive
- **Visual appeal & ease of navigation rating:** ⭐ 4.14 / 5
 
### Key Additions Requested
 
- User profile to track app history and accumulated rewards *(64% of users)*
- Hydration tracker feature *(29% of users)*
 
> **Note:** Minor recommendations involved improving interactive elements on the analytics page. One outlier response was recorded for the majority of questions.
 
---
 
## Visual Design Feedback
 
### Colors, Fonts & Layout
 
**92.9%** of users found the selected colors, fonts, and layout visually pleasing. The color choices were informed by recurring fitness app trends — high-contrast, bright multicolor icons that support user mood and motivation, a concern raised during the Requirement Analysis phase.
 
One user flagged the analytics page as having cluttered elements.
 
### Fitness Application Aesthetics
 
**92.9%** of respondents felt the interface was appropriate for a modern fitness tracking application. One outlier response was recorded and treated as an extreme value.
 
---
 
## User Suggestions / Requests
 
The prototype was well received, with the most common request centered around a user profile feature.
 
| Feature Request | Responses | User % | Sample Quote | Priority |
|---|---|---|---|---|
| User Profile | 9 | 64% | *"I would like to see a profile page where I can see my progress"* | 🔴 Must Have |
| Hydration Tracker | 4 | 29% | *"I need a hydration menu, I need to know how much water I drink"* | 🔴 Must Have |
| Customization | 3 | 21% | *"A profile option that could allow me to maybe change the color theme of the app"* | 🟡 Should Have |
| Socialization / Buddy System | 2 | 14% | *"Maybe a friend system that allows me to add a friend to see their statistics or badges"* | 🟢 Could Have |
